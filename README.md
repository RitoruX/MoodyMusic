## Getting Started

Install dependencies for both web and server:

`npm install`

Run server:

`npm run devStart`

Run web:

`npm run dev`

Visit web application at: <http://localhost:1234>

Visit swagger ui documentation at: <http://localhost:3000/api-docs>
